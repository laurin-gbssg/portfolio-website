Meine Fähigkeiten

Technologien und Sprachen, mit denen ich täglich arbeite

  • HTML

    Die Grundlage jeder Website. Ich strukturiere Inhalte semantisch korrekt mit HTML5 und lege damit das Fundament für zugängliche und gut wartbare Webseiten.

  • CSS

    Mit CSS gestalte ich ansprechende, responsive Layouts. Ich nutze moderne Techniken wie Flexbox, Grid und CSS-Variablen für sauberen, wartbaren Stil-Code ohne Redundanzen.

  • Java

    Java ist eine meiner Kernsprachen. Ich programmiere objektorientiert, nutze Datenstrukturen effizient und setze Algorithmen strukturiert um.

  • SQL

    Mit SQL entwerfe und befrage ich relationale Datenbanken. Ich beherrsche SELECT, JOIN und DDL-Befehle für die strukturierte Verwaltung und Abfrage von Daten.

  • Python

    Python setze ich für Skripting, Datenverarbeitung und kleinere Automatisierungsprojekte ein. Die klare Syntax macht es ideal für schnelle Problemlösungen.
